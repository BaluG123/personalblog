# Balappa Goudi — Portfolio

Personal portfolio for **Balappa Goudi**, React Native developer (4+ years) with 17+ live Play Store apps.

## Stack

- React + TypeScript (Vite)
- Tailwind CSS v4
- Framer Motion

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```

Live site will be at: `https://BaluG123.github.io/personalblog/`
